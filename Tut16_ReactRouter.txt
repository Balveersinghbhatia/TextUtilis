What is React Router?
React Router is a fully-featured client and server-side routing library for React, a JavaScript library for building user interfaces. React Router runs anywhere React runs; on the web, on the server with node.js, and on React Native.

In layment terms
It is a library which is used for routing purpose.
Routing means where to route (send).
It is used for making SPA' Single page application

SPA is an app where all pages are loaded with reloading the page.
Only the components which are changing are loaded.
